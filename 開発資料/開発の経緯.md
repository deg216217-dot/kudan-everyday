# 開発の経緯（まいにち九段ベースボール）

小学4年生（千代田区立九段中等教育学校をめざす）のための学習アプリを、対話しながら段階的に作り込んだ記録です。

## 背景・目的

- 対象：九段中をめざす小4男児。プロスピ（野球ゲーム）とYouTubeが好きで、勉強は苦手・受け身。視覚/聴覚優位、英語と野球が好き。英検3級。
- 将来の目標：MLBの通訳。→ 英語＋野球を続けること自体が訓練になる。
- 九段の適性検査：報告書200＋適性1(国語)200＋適性2(資料)300＋適性3(理数)300＝1000点。配点の約6割が算数・理科・資料系。報告書は小4から加味。すべて独自問題。
- 家庭でできる対策：「今日はどんな一日だった？」「なぜ？」を言葉にする習慣づけが、適性検査（資料を読んで考える・選んで理由を書く）に直結。

## 設計の柱

1. **毎日の習慣化**：5つのデイリークエスト（Duolingo / 計算 / 音読 / 作文 / 理科社会）。
2. **ごほうびループ**：5つ全クリアで、選手カードガチャ＋お金ルーレット（月1,400円ペース）。カードは「球団」図鑑に。
3. **完全オフライン・単体動作**：ネットもログインも不要。データは端末の localStorage に保存。
4. **10歳目線**：難しすぎない問題、身近で答えられるお題、選んでから理由を書く三択。

## バージョンの流れ

- **v1〜v4**：チェックリスト → AIチューター案 → ガチャ報酬 → ライトテーマ化。配色をクリーム×ネイビーに統一。ジャンル選択を廃止し、タップするだけで始められるように。
- **v5**：**完全オフライン化**。AI通信（API）を全廃し、コンテンツを内蔵バンク化（音読20・作文20・理科28）。保存を localStorage に。同じ問題が連続で出ない回転を実装。計算ヒントも内蔵に。
- **v6**：理科社会を**三択化**。「いきなり"なぜ？"」をやめ、手がかり→三択→「なぜ選んだ？」を言語化→本当の答え＋複数の見方、という九段の適性検査と同じ流れに。
- **v6（Gemini連携）**：作文・理科に、書いたものをコピーして**Gemini先生に見せてほめてもらう**フェーズを毎日追加。お手本も残して比較できるように。
- **v7（①見える化）**：計算は種類別、理科はジャンル別に正答率をグラフ表示。「伸びてる・もう一歩・避け気味」を自動コメント。
- **v9（②自動で寄せる）**：苦手な計算の種類・正答率の低い/あまりやらないジャンルを、多めに出題。上限を設けて偏りすぎを防止。新規状態では均等。
- **v10（③弱点リスト＋復習）**：まちがえた問題を記録し、数日後に自動で再出題（復習）。弱点だけを集めた「ニガテこくふくクエスト」専用モードを追加。
- **v11（作文の10歳目線化）**：抽象的すぎた9題（「すごい発明は？」「リーダーに必要なものは？」等）を、自分の生活・経験から答えられるお題に差し替え。
- **v11（ブラッシュアップ）**：10歳としてテストプレイ＋コンサル視点でセルフレビューし、以下を改善。
  - 音読に「もう1話よむ」ボタン（1話でクリアOK、習慣づけ優先）
  - さくせんタブの説明を現状（自動寄せ・弱点復習・Gemini・見える化）に全面更新
  - 計算レベル3を「小4＋ はってん」に正直化（速さ・割合は小5内容のため）
  - 作文を「まず2つうめればOK」に緩和（つまずき防止）
  - 理科の理由入力を「〜だから」まで促す形に
- **v12（公開準備）**：GitHub Pages 公開用にPWA対応（manifest・アイコン・ホーム画面追加メタ）を追加。配布パッケージ化。

## 技術メモ

- 単一HTMLファイル。フォント：Mochiy Pop One + Reddit Sans。
- 保存キー：`kudan-state-v5`（localStorage、メモリ予備つき）。
- コンテンツ：音読20話・作文20題・理科28問（各テーマ＝野球/生き物・宇宙/メカ/歴史）。難しい漢字に《》ふりがな。
- 計算エンジン：全レベル・全種で正の整数のみ（検証済み）。割合は整数になるよう調整。
- ガチャ：架空の選手カード24種（実在IPなし）。お金分布の期待値＝46.7円/回 → 月30日で1,400円（モンテカルロ検証済み）。
- 出題の寄せ：ジャンル頻度の重み付け＋近い再出題の回避（recentキュー）。計算は苦手カテゴリ重み付け＋1セッション最大2問/種類。
- 弱点復習：間違いを記録→3日後にdue→ふだんの出題に自動注入＋専用クエスト。正解でリストから解除。
