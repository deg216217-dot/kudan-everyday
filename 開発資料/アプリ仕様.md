# アプリ仕様（まいにち九段ベースボール）

## 画面構成（タブ）

1. **きょう（⚾）**：今日の5つのデイリークエスト。進捗バー、連勝、貯金、ガチャ。
2. **球団（🃏）**：集めた選手カードの図鑑（レア度別、コンプ率）。
3. **きろく（📅）**：連勝・ホームランの日・貯金・計算レベル、カレンダー、**学習のみえる化**、**ニガテこくふくクエスト**。
4. **メモ（📝）**：子どもが書いた作文・理科の答えと、お手本のログ（保護者用）。
5. **さくせん（🗺️）**：5クエストの説明、アプリが自動でやること、九段中のしくみ、全データ初期化。

## 5つのデイリークエスト

| クエスト | 内容 |
|---|---|
| 🦉 Duolingo | 別アプリで実施。保護者が確認してチェックするだけ。 |
| 🔢 計算 | 小4の問題を5問。レベル自動調整、苦手寄せ、弱点復習あり。 |
| 📖 音読 | 短いお話を1話、声に出して読む。「もう1話よむ」で追加可。青空文庫リンクも。 |
| ✍️ 作文 | 身近なお題を穴うめで意見文に。お手本＋Gemini先生に見せる。 |
| 🔬 理科社会のタネ | 「今日のなぜ？」を、手がかり→三択→「なぜ選んだ？」で考える。 |

5つ全クリア → 選手カードガチャ → お金ルーレット（1日1回）。

## コンテンツ量（内蔵）

- **音読**：20話（野球5・生き物5・メカ5・歴史5）。各約200字＝1分程度。読みとり質問＋花丸お手本つき。
- **作文**：20題。穴うめテンプレート＋穴ごとのヒント＋完成文例2つ。10歳が生活・経験から答えられるお題。
- **理科社会**：28問（各テーマ7問）。考える手がかり＋三択（正解1つ・各選択肢にコメント）＋本当の答え＋複数の見方。
- 難しい漢字には《》でふりがな。

## 自動化ロジック（③まで実装済み）

### ① 見える化（きろくタブ）
- 計算：種類別（たし算/ひき算/かけ算/わり算/□/割合/速さ）の正答率バー。
- 理科：ジャンル別の正答率＋回数。音読/作文：ジャンル別の回数。
- 「🌟いま伸びてる / 💪もう一歩 / 👀避け気味」を自動コメント（各3回以上が目安）。

### ② 本人に寄せる（自動）
- 計算：正答率の低い種類を多めに出題（1セッション最大2問/種類でバランス維持）。
- 理科/音読/作文：正答率の低い・あまりやらないジャンルを多めに（上限2.0倍）。
- 近い再出題は回避。データが無いうちは均等。

### ③ 弱点リスト＋復習
- まちがえた計算問題・理科の三択を記録。3日後に「due（復習どき）」になる。
- ふだんの計算（5問中1問）・理科（約半分の確率）に、復習問題として自動で出る（🔁ふくしゅうバッジ）。
- 正解でリストから解除（🔥ニガテこくふく）。まちがえると再スケジュール。
- 「🔥ニガテこくふくクエスト」：今の弱点だけを連続で練習する専用モード。

- localStorage キー：`kudan-state-v5`。
- 保存内容：今日のチェック、履歴（カレンダー）、計算レベル、種類別/ジャンル別成績、貯金、カード、ガチャ履歴、出題履歴（recent）、弱点リスト（weak）。

- 自傷・危険・不適切な内容は含めない。子どもが安心して毎日触れる設計。
